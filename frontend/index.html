关于 SBTI-Bench

项目介绍

SBTI-Bench 是一个创新的基准测试项目，将真人 SBTI 性格测试系统改造成适用于大语言模型的评测工具。

通过15个心理维度的标准化测试，我们评估不同 LLM 对性格相关问题的反应模式，从而洞察它们的行为倾向和决策模式。

目前已支持 100+ 种 LLM 模型，通过 OpenRouter API 统一接口进行评测。
